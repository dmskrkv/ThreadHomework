Реализовать класс с функцией, которая на вход принимает массив путей к файлам (не более 5),
содержащим числа попадающие в `int`-диапазон , разделенных пробелами.
По окончанию работы функция возвращает максимальное число из всех содержащихся в файлах.

Каждый файл должен обрабатываться отдельных потоком;
* Преобразование из строки в число должно быть реализовано в самой программе;
Предлагаемая сигнатура функции:

int findMax(String[] filePaths) throws Exception;
